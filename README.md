# LTX Sports Shop

Premium kitesurf webshop gebouwd voor Slingshot kites en kiteboarding gear. Het doel was geen standaard e-commerce site, maar een high-end lifestyle merk dat de sfeer van kitesurfen verkoopt.

## Tech stack

- **Next.js 16** (App Router)
- **TypeScript**
- **CSS Modules**
- **GSAP + ScrollTrigger** voor animaties
- **Unsplash** voor productafbeeldingen

## Pagina's

| Pagina | Route |
|---|---|
| Homepage | `/` |
| Kite overzicht | `/kites` |
| Kite productpagina | `/kites/[id]` |

## Lokaal starten

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
